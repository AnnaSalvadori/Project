Homepage

The application developed is capable of reading a dataset containing information about the genome. The data taken as input is represented in a particular file format named ".GFF3". The information read by the software can be processed in different operations that aim to analyze the data and to access it for specific purposes.

The final appearance of the software, from a user’s point of view, is a web-based interface. The right panel contains references to the other available views, that can be accessed by clicking on the interactive buttons.


Active operations

All the active operations are listed and can be selected from the form to process the dataset.
After submission, the page visualizing the executed operation is opened. Some operations
might be inactive; therefore, by clicking on “View all operations” in the “Active operations” page,
the user will be directed to a page that lists all the operations existent.

GFF3 dataset

The input of the software is the human genome annotation file.
The visualized table presents the 9 columns characteristic of the GFF3 format,
in addition to an initial column for the indexes. Considering the size of the data, only the
first ten and the last ten entries are reproduced for better visualization.

Project documentation

The page contains a more in depth description of the software and the algorithms implemented in it.
The CRC cards and the UML class diagram are provided for a more efficient explanation.



At the end of each page is present a clickable "Home" button, that takes the user to the homepage.
