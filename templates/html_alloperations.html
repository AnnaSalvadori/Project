Full list of operations

  • Info:
    returns the name and the type of each column of the dataset


  • Sequence ID:
    returns the list of unique sequence IDs available in the dataset


  • Type Operation:
    returns the unique types of operations available in the dataset


  • Count Features:
    returns the number of features provided by the same source


  • Count Entries:
    returns the number of entries for each type of operation


  • Get Entire Chromosomes:
    returns a new dataset containing only the information about entire chromosomes (source: GRCh38)


  • Count Supercontigs:
    calculates the fraction of unassembled sequences (supercontigs) over entire chromosomes


  • Entries from sources ensembl, havana, and ensembl_havana:
    returns a new dataset containing entries from the sources ensembl, havana, and ensembl_havana


  • Count Entries from ensembl, havana, and ensembl_havana:
    returns the number of entries for each type of operation in the dataset containing only entries from the sources ensembl, havana, ensembl_havana


  • Gene Names:
    returns the names of genes in the dataset containing only entries from the sources ensembl, havana, ensembl_havana
